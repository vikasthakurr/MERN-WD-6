CSS Units

CSS units define how to measure dimensions, spacing, and font sizes. They are categorized into relative and absolute units.

Absolute Units

These units have a fixed size and are not affected by other elements or screen sizes.

  • px (pixels): A single pixel of the display.
    • Width: 100px, Height: 50px
  • cm (centimeters): Measurement in centimeters.
    • Width: 5cm, Height: 2cm
  • mm (millimeters): Measurement in millimeters.
    • Width: 20mm, Height: 10mm
  • in (inches): Measurement in inches (1in = 2.54cm).
    • Width: 2in, Height: 1in
  • pt (points): 1/72 of an inch.
    • Width: 100pt, Height: 50pt
  • pc (picas): 1 pica = 12 points.
    • Width: 10pc, Height: 5pc

Relative Units

These units are relative to another value, such as the font size of the parent or the viewport.

  • % (percentage): Relative to the parent element's size.
    • Width: 50%, Height: 50%
  • em: Relative to the font size of the element's parent.
    • Font-size: 2em
  • rem (root em): Relative to the root element's font size.
    • Font-size: 1.5rem
  • vw (viewport width): 1% of the viewport's width.
    • Width: 50vw, Height: 10vh
  • vh (viewport height): 1% of the viewport's height.
    • Width: 20vw, Height: 50vh
  • vmin: 1% of the smaller of the viewport's width or height.
    • Width: 10vmin, Height: 10vmin
  • vmax: 1% of the larger of the viewport's width or height.
    • Width: 10vmax, Height: 10vmax
  • ch: Relative to the width of the "0" character in the element's font.
    • Width: 30ch
  • ex: Relative to the height of the element's font's "x" character.
    • Height: 2ex
